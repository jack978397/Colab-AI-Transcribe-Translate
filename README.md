# 🚀 Colab AI 影音轉錄與批次翻譯系統

這是一個整合了 **OpenAI Whisper (辨識)** 與 **Google Gemini 3.1 Pro (翻譯)** 的高效能影音處理解決方案。專為需要高精準度字幕、口語化翻譯，以及有批次處理需求的開發者與創作者設計。

---

## 🌟 專案亮點
- **雲端運算**：免去地端顯示卡不足的困擾，直接利用 Google Colab 的 GPU 資源。
- **雙模組解耦**：將「轉錄辨識」與「獨立翻譯」拆分為兩個獨立系統，靈活應對不同情境。
- **專業級翻譯**：透過 Gemini 3.1 Pro 實現「資深老師」風格的自然口語翻譯，非一般生硬的機器翻譯。
- **反幻覺機制**：強化版辨識邏輯，有效解決 Whisper 在寂靜處或噪音中重複跳針的問題。

---

## 🛠️ 工具說明

### 1. 🎬 [1_Whisper_Transcription_Hub.ipynb]
**功能**：高品質影音辨識工作站
- 支援直接輸入 **YouTube 網址** 或上傳本地影音檔。
- 使用 Whisper `large` 模型，兼顧速度與準確率。
- **自然斷句優化**：自動處理過長的字串，產出符合人眼閱讀習慣的 SRT。
- **即時校對 UI**：內建「批次取代」與「編輯區」，在翻譯前先行修正關鍵字。

### 2. 🌐 [2_Gemini_Batch_Translator.ipynb]
**功能**：AI 專業字幕批次翻譯師
- **批次處理**：支援一次上傳多個 SRT 檔案進行同步處理。
- **實時回饋**：內建實時日誌系統，每完成一個檔案即刻更新狀態與下載連結。
- **動態語言支援**：中、英、日三語互轉，自動適應對應的語言風格與角色設定。
- **風格規則控制**：強制維持時間軸、保留括號專用稱號、句尾自定義去標點。

## 🚀 快速開始

1. **獲取 API Key**：前往 [Google AI Studio](https://aistudio.google.com/app/apikey) 免費申請 Gemini API Key。
2. **打開筆記本**：
    - 點擊本倉庫中的 `.ipynb` 檔案。
    - 點擊上方的 **「Open in Colab」** 按鈕。
3. **執行環境**：
    - 在 Colab 點擊選單「執行階段」 -> 「變更執行階段類型」 -> 選擇 **T4 GPU** (轉錄工具必備)。
    - 依序執行儲存格即可開啟 Web UI。
